## 📌 Sobre mim

Sou **Amom Reis**, estudante de **Engenharia de Software na UEPA (3º ano)** e desenvolvedor **Back-end focado em Java e Spring Boot**.

Tenho grande interesse em construir sistemas **robustos, escaláveis e bem estruturados**, principalmente soluções voltadas para **produtos SaaS e automação de processos para pequenos negócios**.

Atualmente estou desenvolvendo um **MicroSaaS**, com foco em:

- confiabilidade
- escalabilidade
- experiência do usuário
- crescimento em parceria com o cliente

Busco evoluir constantemente como desenvolvedor e contribuir em projetos que **gerem impacto real**.
